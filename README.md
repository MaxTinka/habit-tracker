# Habit Tracker – Web App

## Description
A beautiful and intuitive habit tracking web application that helps you build better habits. Track daily habits, maintain streaks, and visualize your progress over time.

## Live Demo
[View Live Demo](https://maxtinka.github.io/habit-tracker)

## Features
- ➕ Add unlimited habits with custom colors
- ✅ Mark habits as complete for each day
- 🔥 Track current and best streaks
- 📊 View statistics (total completions, completion rate)
- 📅 7-day week view for easy tracking
- 💾 LocalStorage persistence (data saves automatically)
- 📤 Export data to JSON file
- 🎨 Modern, responsive design with gradients
- 📱 Mobile-friendly interface

## Technologies Used
- **HTML5** – Structure and layout
- **CSS3** – Styling, gradients, animations
- **JavaScript (ES6)** – Dynamic rendering, localStorage, calculations
- **Font Awesome** – Icons
- **Google Fonts** – Inter font family

## How to Use
1. Add habits using the input field and color picker
2. Click on day buttons to mark habits as complete
3. Green checkmarks show completed days
4. View your streaks and statistics at the top
5. Data saves automatically in your browser
